# Personal Assignment 1 – Sistem Data Mahasiswa

Program Java sederhana untuk mengelola data mahasiswa (OOP, enkapsulasi, method).

## Cara Menjalankan

Buka terminal di folder `Personal Assignment 1`, lalu:

### Soal 1 – Class dan Object

Menampilkan data 5 mahasiswa.

```bash
javac soal1.java
java soal1
```

Atau langsung (Java 11+):

```bash
java soal1.java
```

### Soal 2 – Enkapsulasi dan Update IPK

Update IPK mahasiswa berdasarkan NIM.

```bash
javac soal2.java
java soal2
```

Saat dijalankan, masukkan NIM (misal: `2440002`) dan IPK baru (misal: `3.25`).

### Soal 3 – Predikat Akademik

Lihat data mahasiswa beserta status dan predikat berdasarkan NIM.

```bash
javac soal3.java
java soal3
```

Saat dijalankan, masukkan NIM (misal: `2440003`) untuk melihat info dan predikat.
